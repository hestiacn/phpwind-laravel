# 主题切换功能

## ✨ 支持的三种模式

### ☀️ 明亮模式

- **特点**：白底黑字，清晰明快
- **适用**：白天使用，光线充足的环境
- **视觉效果**：紫色渐变卡片，白色背景

### 🌙 暗黑模式

- **特点**：深色背景，护眼舒适
- **适用**：夜间浏览，光线较暗的环境
- **视觉效果**：深色卡片，低对比度文本

### 🔄 自动模式

- **特点**：根据时间自动切换
- **规则**：
  - 6:00 - 18:00：明亮模式
  - 18:00 - 6:00：暗黑模式
- **依据**：使用您浏览器的本地时间

## 🎯 如何切换主题

### 方法一：点击主题按钮

页面右上角有一个主题切换按钮，点击即可选择模式：

![主题切换按钮](/images/theme-switcher.png)

1. 点击当前主题图标（☀️/🌙/🔄）
2. 从下拉菜单中选择想要的模式
3. 页面立即切换，无需刷新

### 方法二：个人设置

如果您已登录，可以在个人中心永久保存主题偏好：

1. 点击右上角头像 → **个人中心**
2. 找到 **主题设置** 选项
3. 选择您的偏好模式
4. 点击 **保存设置**

### 方法三：系统默认

未登录用户或未设置偏好的用户：
- 默认使用 **自动模式**
- 主题偏好会保存在浏览器中
- 下次访问自动生效

## 📱 响应式设计

主题切换在所有设备上都能完美工作：

| 设备 | 显示效果 |
|------|----------|
| 桌面端 | 完整的主题切换器 |
| 平板 | 折叠菜单中的主题选项 |
| 手机 | 底部导航或菜单中的主题按钮 |

## ❓ 常见问题

**Q: 主题切换会影响所有页面吗？**  
A: 是的，主题设置会应用到整个网站的所有页面。

**Q: 未登录用户能保存主题设置吗？**  
A: 可以！主题偏好会保存在浏览器的本地存储中，下次访问自动生效。

**Q: 自动模式的时间依据是什么？**  
A: 使用您浏览器的本地时间判断。如果您在不同时区访问，自动模式会根据您当地的日出日落时间切换。

**Q: 可以自定义主题颜色吗？**  
A: 目前支持预设的明亮/暗黑模式，暂不支持自定义颜色。

## ⚙️ 技术实现

主题切换通过以下方式实现：

```css
:root {
  --bg-primary: #ffffff;
  --text-primary: #333333;
  /* ... 其他变量 */
}

[data-theme="dark"] {
  --bg-primary: #1a1a1a;
  --text-primary: #f0f0f0;
  /* ... 暗黑模式变量 */
}
```

```javascript
// 切换主题
function setTheme(theme) {
  document.documentElement.setAttribute('data-theme', theme);
  localStorage.setItem('theme', theme);
}
```
