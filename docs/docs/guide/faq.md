# 常见问题 (FAQ)

## 👥 账号相关

### 如何注册账号？

1. 点击首页右上角的 **注册** 按钮
2. 填写用户名、邮箱和密码
3. 点击 **注册** 完成
4. 前往邮箱激活账号

### 忘记密码怎么办？

1. 点击登录页面的 **忘记密码** 链接
2. 输入您的注册邮箱
3. 查收密码重置邮件，按提示操作

### 可以修改用户名吗？

目前不支持修改用户名。注册时请谨慎选择。

## 🎨 功能相关

### 农历节日准确吗？

非常准确！我们使用了专业的农历计算库，支持1900-2200年的农历转换，包括二十四节气。

### 主题切换支持哪些模式？

支持三种模式：明亮模式、暗黑模式、自动模式（根据时间切换）。

### 哀悼日模式会自动开启吗？

是的。在5月12日和12月13日，网站会自动启用灰度模式。

### 多语言支持哪些语言？

目前支持中文、英文、日文、韩文、德文、法文、俄文、越南文、印地文等。

## 💻 技术相关

### 论坛基于什么技术构建？

基于 Laravel 12 重构，前端使用 Blade 模板 + Bootstrap 5。

### 数据从哪里来？

数据来自原 phpwind 论坛数据库，通过迁移脚本导入。

### 开源吗？可以贡献代码吗？

是的，完全开源！欢迎访问我们的 [GitHub 仓库](https://github.com/hestiacn/phpwind-laravel) 贡献代码。

## 🚀 使用技巧

### 快速切换主题

在页面右上角点击主题图标，可以快速在明亮/暗黑/自动模式之间切换。

### 快速切换语言

在页面右上角点击语言标识，可以快速切换语言。

### 查看农历信息

页面左上角会显示当天的农历日期和节气信息。
