# 农历功能说明

## 📅 功能概述

论坛会根据访问者的**本地时间**自动判断当天是否是农历节日、公历节日或二十四节气，并显示相应的图标和提示信息。

- **中文用户**：直接显示农历日期和节日名称
- **非中文用户**：显示提示信息"农历是中国传统历法"，鼠标悬停时显示具体农历

## 🔧 技术实现

论坛使用**访问者的本地时间**进行判断，确保全球用户看到符合自己时区的节日信息：

```javascript
// 伪代码示例
const userDate = new Date();
const holiday = detectHoliday(userDate);
if (holiday) {
    showHolidayIcon(holiday.icon);
}
```

## ❓ 常见问题

**Q: 为什么我看到的节日和实际日期不符？**  
A: 请检查您设备的系统时间设置是否正确。

**Q: 农历节日准确吗？**  
A: 非常准确！我们使用了专业的农历计算库，支持1900-2200年的农历转换。

**Q: 可以自定义节日吗？**  
A: 管理员可以在后台配置自定义节日。
