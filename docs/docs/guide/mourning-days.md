# 哀悼日模式

## 🕯️ 功能说明

在国家规定的哀悼日，网站会自动启用**灰度模式**，以表达对遇难同胞的深切哀悼。

## 📅 哀悼日列表

| 日期 | 名称 | 说明 |
|------|------|------|
| 5月12日 | 汶川地震纪念日 | 2008年汶川大地震纪念日 |
| 12月13日 | 南京大屠杀纪念日 | 南京大屠杀死难者国家公祭日 |

## 🎨 视觉效果

启用哀悼日模式时：

- 整个网站变为**灰度显示**
- 所有彩色元素（图标、图片、按钮）转为黑白
- 页面顶部显示哀悼横幅
- 横幅中包含哀悼信息和一个 🕯️ 图标

## 🔍 如何判断

论坛会根据**访问者的本地时间**自动判断当天是否是哀悼日：

- 如果您的本地时间是5月12日，网站自动启用灰度模式
- 如果您的本地时间是12月13日，网站自动启用灰度模式
- 其他日期正常显示

## 💻 技术实现

```css
/* 灰度模式 CSS */
body.mourning-mode {
  filter: grayscale(1);
  -webkit-filter: grayscale(1);
  transition: filter 0.5s ease;
}
```

```javascript
// 检测哀悼日
const today = new Date();
const month = today.getMonth() + 1;
const day = today.getDate();

if ((month === 5 && day === 12) || (month === 12 && day === 13)) {
  document.body.classList.add('mourning-mode');
  showMourningBanner();
}
```

## ❓ 常见问题

**Q: 为什么我的网站变成灰色了？**  
A: 今天是国家规定的哀悼日，网站自动启用灰度模式以表达哀悼。

**Q: 可以手动关闭灰度模式吗？**  
A: 哀悼日当天无法手动关闭，这是为了尊重和纪念。您可以通过点击横幅右上角的关闭按钮暂时隐藏横幅，但网站的灰度效果会保持。

**Q: 哀悼日模式会影响网站功能吗？**  
A: 不会！只有视觉效果发生变化，所有功能（发帖、评论、登录等）仍然正常使用。
